As you progress through your course, it is crucial to regularly assess your understanding and capabilities against the learning outcomes. This exercise is designed to help you reflect on your learning journey, identify areas where you may need to deepen your knowledge or skills, and develop a plan for improvement. Use this checklist to evaluate your understanding of the lessons covered in Topic 7. If you don't feel confident about any of these concepts/tasks, revisit the relevant lecture videos, readings and activities to help you consolidate your knowledge. 1. Understanding Linear Data Structures I can define and explain what linear data structures are. I can describe the operations of linear data structures using pseudocode (e.g., insertion, deletion, traversal). I understand the differences between arrays and linked lists as linear data structures. 2. Linked Lists I can explain the structure and components of a linked list (e.g., nodes, pointers). I can discuss the process of creating, inserting, deleting, and traversing elements in a linked list. I successfully implemented a sorted linked list in code. 3. Stacks I can explain the stack data structure and its LIFO (Last In, First Out) principle. I can explain how to implement a stack using arrays. I can explain how to implement a stack using linked lists. I can write pseudocode for common stack operations such as push, pop, and peek. 4. Queues I can explain the queue data structure and its FIFO (First In, First Out) principle. I can explain how to implement a queue using arrays. I can explain how to implement a queue using linked lists. I can write pseudocode for common queue operations such as enqueue, dequeue, and peek. 5. General Application I can identify real-world problems where stacks and queues are appropriate solutions. I feel confident in switching between array-based and linked list-based implementations for stacks and queues. I can explain the advantages and disadvantages of using linked lists over arrays for specific operations or scenarios. 6. Reflection I can identify areas where I need more practice or clarification. I have reviewed and understood all examples and exercises provided during this topic.

As you progress through your course, it is crucial to regularly assess your understanding and capabilities against the learning outcomes. This exercise is designed to help you reflect on your learning journey, identify areas where you may need to deepen your knowledge or skills, and develop a plan for improvement. Use this checklist to evaluate your understanding of the lessons covered in Topic 8. If you don't feel confident about any of these concepts/tasks, revisit the relevant lecture videos, readings and activities to help you consolidate your knowledge. 1. Understanding Tree Structures I can explain the basic structure and components of a tree, including nodes, edges, root, and leaves. I understand the differences between a generic tree, binary tree, and binary search tree. I can implement a basic tree structure using pseudocode or a programming language. 2. Binary Tree Traversals I can describe the three primary types of binary tree traversal: In-order traversal Pre-order traversal Post-order traversal I can trace the sequence of nodes visited during each type of binary tree traversal using pseudocode. I can identify appropriate scenarios for using each traversal method. 3. Binary Tree Operations I understand the properties that define a binary search tree (BST). I can describe and implement the process of building a BST from a set of values. I can trace and explain the steps of a search operation in a BST using pseudocode. I can modify a BST by: Inserting new nodes while maintaining BST properties. Deleting nodes (leaf nodes, nodes with one child, and nodes with two children) and maintaining BST properties. 4. General Problem-Solving I can identify problems that can be efficiently solved using binary trees or binary search trees. I can apply traversal and search techniques in real-world scenarios or coding tasks. I can debug and optimize tree-related algorithms for improved performance. 5. Reflection I feel confident in my understanding of trees and binary search trees. I can identify areas where I need further practice or clarification.

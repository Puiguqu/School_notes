That is, a recursive algorithm that executes a finite number of recursive calls. Let's look at a modified version of the recursive printing algorithm we worked with in the last lecture. You can see several changes in that function. First, the function now receives an integer value as an input argument. Secondly, there's an instruction that it stops the recursion. If n is equal to zero, we stop making a recursive call and we simply return. This part of the code is known as the base case. To avoid infinite recursion, one condition you must meet is having at least one base case in your recursive algorithm. Thirdly, when we make the recursive call, we have to provide an input argument. With every recursive call, that input argument gets closer to the base case. In that way, we make sure that we are going to reach the base case at some point and thus stop the recursion. So every world build recursive algorithm should include at least one base case and one recursive call with an input argument getting closer to the base case. Now let's trace the new version of this algorithm to find out what task it performs.

The technique can be summarized as doing a small part of the job and then delegating the rest. In delegating the rest do not worry about the details of how the task will be performed. Let's now apply this technique to write a recursive linear search algorithm. This algorithm solves the problem of determining whether a number is present in an array or not. If the number is in the array, the Boolean value true is returned. Otherwise, the Boolean value false is returned. The algorithm receives as input arguments, the array, the number of elements of it, and the number to search for. Now, let's think how to solve this problem recursively. First, we have to decide what part of the job we are going to do and next what part we are going to delegate. The small part we are going to do is checking whether the number X is in one of the positions. We can check any position, But since we are given the value n it's easier to check whether number X is in the last position of the array we can do that by writing this if statement if x is in the last position of the array then return true If not, we are going to delegate the rest of the work by giving the remaining positions of the array to someone else that someone else is our recursive call. We do that by writing an instruction outside of the if statement making the recursive call. When making the recursive call, we are delegating the rest of the work by giving a smaller version of the problem to someone else. As it is now, our recursive function does not have a base case. In this particular problem the base case occurs when we have finished checking all the repositions. If we get to the case where n equals minus one, it means we have no more positions to check and thus number x is not in the array. So we add the base case by writing this if statement. Now, we have a base case, our small part of the total job and the part we delegate to someone else. Our recursive algorithm is now complete and the only task left to us is tracing it to make sure it's working as needed. That's your job now.

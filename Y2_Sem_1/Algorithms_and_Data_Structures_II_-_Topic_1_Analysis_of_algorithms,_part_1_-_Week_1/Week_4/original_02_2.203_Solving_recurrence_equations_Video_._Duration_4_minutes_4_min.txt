In the last lecture, we found out an expression for the running time of the recursive algorithm that calculates the factorial of N. We learned that such expression, where the running time of N depends on the running time of N minus one is known as a recurrence equation. Now, the problem with having a recurrence equation, is that we do not have an explicit expression for the running time. We cannot plot this function. Thus, we need to solve the expression. To do so, we need to do two things. First, find a value of N for which T(N) is known. Usually, this is easy to do with the running time of the best-case. Second, expand the right side of the recurrence equation until you can't replace the known value of T(N) on it. We will now solve the recurrence equation of the recursive algorithm that computes the factorial of N using these two steps. To find a value of N for which T(N) is known, we have to go back to the algorithm and find the running time for the best case. In the best case, the algorithm evaluates the condition of the if statement. The condition is true, and then the return instruction is executed. Both instructions are executed in constant time. This case occurs when N is equal to one. So we can say that T(1) one is equal to a constant C. Now, we have to expand the recurrence equation until we can't replace the noun value on its right side. To do so, we expand the expression of T(N-1) using the definition of T(N). T(N) is equal to C5 plus T(N-1). Therefore, T(N-1) is equal to C5 plus T(N-1-1), which is T(N-2). Now, we expand the expression T(N-2), applying the same method. You can start seeing a pattern here. As we keep on expanding the right side of the recurrence equation, if the constant C5 is multiplied by k, then, the argument of T(N) gets decreased my k units, that is, every expansion makes the argument of T(N) smaller and smaller. In fact, when k is equal to N minus one, we know the value for the right side of the recurrence equation. T(N-N-1) is equal to T(1), which we already know is equal to a constant value. Thus we have solved the recurrence equation now. T(N) is no longer described in terms of itself with a smaller argument. We can write T(N) in a simplified way and we can see that T(N) grows linearly with N. Now that we have solved the recurrence equation and know that T(N) grows linearly with N, we can do an asymptotic analysis and say that T(N) is O(N), O(N^2) and O of all the functions that grow faster than N. T(N) is omega of N, omega of log N and Omega of all the functions that grow slower than N. T(N) is theta n.

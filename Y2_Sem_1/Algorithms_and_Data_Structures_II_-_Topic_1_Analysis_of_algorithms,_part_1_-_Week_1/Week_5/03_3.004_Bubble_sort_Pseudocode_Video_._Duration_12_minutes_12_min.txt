**Introduction to Bubble Sort**

The lecturer explains that today they will be reviewing the Bubble Sort algorithm, which is a simple sorting algorithm that works by repeatedly iterating through a list of elements and swapping adjacent elements if they are in the wrong order.

**Bubble Sort Pseudocode**

The lecturer provides the pseudocode for the Bubble Sort algorithm:
```
repeat
  for each element i in the list
    if element i is greater than element i+1
      swap element i and element i+1
  end repeat
end while
return sorted list
```
The lecturer explains that this pseudocode represents the basic steps of the Bubble Sort algorithm:

1. Repeat until no more swaps are needed.
2. For each element in the list, check if it is greater than the next element.
3. If the current element is greater, swap it with the next element.
4. Return the sorted list.

**Example Walkthrough**

The lecturer provides an example of how Bubble Sort works on a sample list: `[5, 2, 8, 1, 9]`.

3. The list now looks like `[2, 5, 8, 1, 9]`.
4. The algorithm continues to iterate through the list, finding no more swaps needed.

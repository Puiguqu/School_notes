## Step 1: Analyze the input arguments of the merge function
The merge function takes two arrays, `l` and `r`, as its input. These arrays contain already sorted elements from the original array `a`.

## Step 2: Identify the three tasks performed by the merge function
The three tasks are:

* Copying the already sorted elements between `l` and `mid` to a new array called `l`.
* Copying the already sorted elements between `mid + 1` and `h`, to a new array called `r`.
* Comparing the first element of `l` to the first element of `r`, whichever is lowest, gets copied back in the corresponding position in the original array `a`.

## Step 3: Write the pseudocode for each task

  l.append(a[i])

## Step 4: Combine the pseudocode for all tasks
Here is the complete pseudocode:
```
FUNCTION merge(l, mid, h)
  CREATE a new array r
  r = [ ]
  FOR i FROM mid+1 TO h DO
    r.append(a[i])
  END FOR
  
  IF length(r) > 0 THEN
    a[mid+1] = merge(l, mid, length(r)-1)
  ELSE
    a[mid+1] = NULL
  END IF
  
  CREATE a new array l2
  l2 = [ ]
  FOR i FROM 0 TO mid-1 DO
    l2.append(a[i])
  END FOR
  FOR i FROM 0 TO min(length(l), length(r)) DO
    IF l2[i] <= r[i]
    THEN
      a[mid+i+1] = l2[i]
    ELSE
      a[mid+i+1] = r[i]
    END IF
  END FOR
  
  RETURN
END FUNCTION
```

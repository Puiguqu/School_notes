First, let's analyze its structure as a recursive algorithm. A recursive algorithm can be recognized easily, because inside its code you find one or more calls to the same algorithm. In this case, Merge sort calls itself twice. The first call must work on the section of the array that starts at position l and finishes at position mid, that is on the left half of the section of the array between l and h. The second call works on the section of the array that starts at position mid+1 and ends at position h. That is, on the right part of the section of the array between l and h. Notice that in this image, the positions l, mid, and h are just possible values. As the execution of the algorithm progresses, they might point to different positions in the array, although the relative order remains. Now, a recursive call should also include at least a base case. That is, a condition where the recursion stops. In this implementation, the base case is implicit in the non-written else part of the if statement. That is, if the condition l is less than h is met, then the recursion continues. But if this condition is not met, the recursive function simply returns and recursion stops. Now, let's understand what part of the code is in charge of what task. We have already seen that this "if statement" is in charge of deciding whether the recursion stops or continues. The first instruction inside the if statement calculates the middle position between l and h. Therefore, the first time the Merge sort function is called, mid stores the position of the element in the middle of array A. , we make a first recursive call to Merge sort. Analogously, when we make the second recursive call to Merge sort with input arguments A, mid+1 and h, the right half of the array is sorted. This instruction is the core of the algorithm. Here is where the sorting is actually carried out. The function merge receives the array and generates two extra arrays, the array l with the sorted left half of A and the array R with the sorted right half of A. So in summary, Merge sort first checks that the recursion can continue.

This study guide provides a detailed list of the skills, concepts, and knowledge required to successfully complete the Coursework 1 Assignment. Checklist of required skills and knowledge 1. Object-Oriented Programming Concepts Classes and Objects: Definition and declaration of classes. Instantiation of objects. Encapsulation Use of private and public access specifiers. Getter and setter methods. Inheritance: Creating subclasses from superclasses. Polymorphism: Method overriding and overloading. Abstract Classes and Interfaces Definition and implementation. 2. Data Handling and Computation Standard Library Usage Proficient usage of standard libraries such as <vector>. File Handling Reading and writing data to and from files. Data Structures Use of vectors, arrays, and other structures to store and manage data efficiently. Candlestick Data Calculation Computation of Open, High, Low, and values from raw data (i.e., temperature). Algorithm Implementation Creating efficient algorithms to process large datasets. 3. Text-Based Visualization Basic Text Manipulation Creating text-based representations of graphical elements. Plot Implementation Designing text-based plots (e.g., candlestick charts). Character Usage Selection of appropriate characters for representing elements of the plot (e.g., ‘-’ for line, ‘|’ for stalk). 4. Data Filtering and Analysis Data Filtering Techniques Implementing suitable filter options (i.e., date range, country, and temperature range) to allow users to explore and analyse data. Graphical Data Representations Creating text-based graphs from filtered data. 5. Prediction Models Predictive Modeling Understanding and implementing different prediction models for time-series data. Data Analysis Methods Basic statistical methods for predicting future data from historical trends. Justification and Explanation Providing explanatory descriptions and justifications for chosen prediction methods. Code Implementation Writing and explaining code for the prediction models. 6. Code Quality and Documentation Code Style ing best practices for indentation, naming conventions, and readability. Error Handling Implementing robust error handling to manage unexpected inputs or conditions. Modular Programming Writing modular and reusable code for different parts of the assignment. To effectively prepare for the upcoming Coursework 1 Assessment, focus on mastering the conceptual understanding of OOP principles, practical skills in handling and processing data, creating visualizations, and implementing predictive models. Additionally, pay attention to maintaining code quality, documenting clearly, and effectively communicating your work.

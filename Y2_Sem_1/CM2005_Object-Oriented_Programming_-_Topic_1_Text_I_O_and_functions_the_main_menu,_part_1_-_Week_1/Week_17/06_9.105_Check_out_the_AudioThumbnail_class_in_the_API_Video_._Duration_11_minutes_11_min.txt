**Section 9.105: Check out the AudioThumbnail class in the API**

The tutor explains that the AudioThumbnail class is used to load and display thumbnails for audio files. They demonstrate how to use this class to load a thumbnail for a selected audio file.

```cpp
// Load the thumbnail
AudioThumbnail thumbnail;
thumbnail.load("path/to/audio/file.wav");

// Get the image from the thumbnail
JUCE::Image img = thumbnail.getThumbnailImage();
```

**Section 9.107: Draw the thumbnail**

The tutor demonstrates how to draw the thumbnail using the JUCE image library.

```cpp
// Create an image view
JUCE::ImageView imageView;
imageView.setColour(Juce::Image::ColourId::BackgroundColour, JUCE::Colour(0x000000));

// Get the image from the thumbnail
JUCE::Image img = thumbnail.getThumbnailImage();

// Draw the image
imageView.drawImage(img);
```

**Section 9.108: Implement a ChangeListener to make sure it draws automatically**

The tutor explains that a ChangeListener is needed to detect changes in the waveform display and redraw it automatically.

```cpp
// Create a ChangeListener
WaveformDisplay::ChangeListener listener;
listener.addChangeListener(WaveformDisplay::ChangeType::RangeChanged, [this] { 
    this->draw();
});

// Get the range of the waveform display
float range = display.getRange();

// Draw the waveform
display.draw(range);
```

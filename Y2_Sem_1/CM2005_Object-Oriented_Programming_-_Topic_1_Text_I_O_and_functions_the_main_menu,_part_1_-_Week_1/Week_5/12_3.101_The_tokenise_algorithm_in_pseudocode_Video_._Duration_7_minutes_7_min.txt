**Topic 3: Tokenise Algorithm in Pseudocode**

The topic focuses on the tokenise algorithm, which is responsible for breaking down text into individual tokens (e.g., words or phrases). The pseudocode version is presented as:

```
FUNCTION tokenise(text)
  input = text
  output = empty array
  WHILE input ≠ empty string
    token = first word in input
    output.append(token)
    remove first word from input
  ENDWHILE
  RETURN output
ENDFUNCTION
```

The pseudocode provides a simple example of how the algorithm works, using a while loop to iterate through each character in the input text and extract individual tokens.

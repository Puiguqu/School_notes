**WHAT IS MOCHA?**

Mocha is a unit testing framework that allows you to write tests for your JavaScript code. It's designed to make it easy to write and run tests, so you can catch bugs early on in the development process.

**CHAI: A COMPANION MODULE TO MOCHA**

But Mocha isn't just about writing tests - it's also about using a companion module called Chai to specify your tests. Chai is a BDD (Behavior-Driven Development) framework that makes it easy to write concise and readable tests.

**RUNNING THE TESTS**

To run the tests, we'll use the Mocha command-line tool. This will execute our test file and report any failures or errors.

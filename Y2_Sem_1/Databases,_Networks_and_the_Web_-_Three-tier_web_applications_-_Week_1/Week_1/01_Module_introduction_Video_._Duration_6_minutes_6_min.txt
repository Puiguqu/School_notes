The module covers the theory and practical skills required to build dynamic web applications that utilize modern web and internet infrastructure. The learning objectives are:

1. Understand the concepts of the relational database model.
2. Implement relational database models.
3. Query data from relational database models.
4. Understand the concepts of web architectures.
5. Build database-driven dynamic web applications.
6. Understand how computer networks work.

The module is divided into 22 topics, covering three-tier web applications, Node.js and Express, template engines, forms, relational databases, database design, security, and networking concepts. The topics are designed to provide a hands-on understanding of how databases, networks, and the web underpin modern Internet applications.

Some key concepts covered in the module include:

* Three-tier web applications
* Node.js and Express
* Template engines (EJS)
* Relational databases and SQL language
* Database design and schema
* Security measures for web applications
* Networking concepts and protocols (HTTP, etc.)

The module aims to provide students with practical skills and knowledge to build complex modern web applications like the ones used in daily life.

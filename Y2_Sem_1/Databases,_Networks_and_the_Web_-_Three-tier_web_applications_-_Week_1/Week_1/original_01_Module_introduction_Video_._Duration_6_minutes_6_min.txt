Some screen readers may require using CTRL in conjunction with the alt key Hello and welcome to Databases, Networks and the Web. My name is Llewellyn Fernandes and together with my colleague Ella, we will give you an understanding of the theory and practical skills required to build dynamic web applications. That is, applications that use modern web and internet infrastructure to deliver data-driven interactive applications. The learning objectives for this module can be summarized as number one, understand the concepts of the relational database model. Number two, implement relational database models. Number three, query data from relational database models. Number four, understand the concepts of web architectures. Five, build database-driven dynamic web applications. And six, understand how computer networks work. In topic one, we will look at the concept of three-tier applications. These are applications where rather than having a single process or monolithic application, the application is split into layers with a presentation tier, application tier and data tier. With the application components communicating between layers using well-defined protocols such as HTTP and SQL. In topic two, you will set up a development environment where you can start building web applications using a server-side JavaScript runtime environment called Node.js. You will use a framework called Express to organize your applications so they can respond to different user requests. In topic three, you will take your web applications further using a templating engine called EJS. This will allow you to take data such as this data about Charlie and his purchases and insert it into a webpage. This lays the foundations for data-driven web applications. In topic four, you will learn how to create forms such as this registration form. Using this form, a user can enter data and that data can be accessed from your web applications. Topic five introduces you to the concept of relational databases. You will see how to create a database and how to create table structures within the database. For example, this table which matches the structure of the form we saw earlier. You will learn how to insert data into the table structure using the SQL language. So the data from a JavaScript object, for example, with a name and email address can be processed to generate a SQL statement that inserts the data into the corresponding database table. You will also learn how to retrieve data from the database, i.e. how to write SQL statements that pull data from a database table into an object in your application. In topic six, you will see how your database can be linked to your web application so that data can be extracted from the database into your application tier and can be presented in a webpage. Topic seven will explore the important concepts in database design and you will learn how to create database schemas involving multiple related tables. For example, linking customers and shops to the purchases made. Furthermore, you will learn how to retrieve data from the connected tables using SQL. SQL and relational databases provide a rich set of functionality. So in topic eight, you will explore some of the more complex data access scenarios such as aggregating data, for example, summing the quantity using the sum function or using one query to select data that is then processed by another query to give a result. Topic nine will explore the important concept of security in web applications and databases, looking at how to secure the data stored in our database and when the data is being transferred over a network or between layers in our application. Topics one to nine focused on building applications that utilized a network to connect things like web servers and browsers. Topic ten looks under the covers of this network. So when you have two computers and you connect them together on a network, what are the mechanisms by which that data is transferred over the network? And how do common networking protocols utilize those mechanisms? You will find this a very hands-on module with lots of lab exercises that should give you a good understanding of how databases, networks, and the web underpin modern Internet applications. The skills you gain will allow you to go on and build more complex modern web applications like the ones you use in your daily lives.

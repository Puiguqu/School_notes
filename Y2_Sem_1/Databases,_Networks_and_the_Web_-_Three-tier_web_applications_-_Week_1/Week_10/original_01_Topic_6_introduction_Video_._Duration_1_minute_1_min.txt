In this topic you will have the chance to practice and review creating and querying one more database example in MySQL Shell. The database example is about storing the menu items of a restaurant in a database. Then you will be introduced to the update and delete operations in SQL and you will be able to test and run those SQL statements on a restaurant menu example in MySQL Shell. The concept of primary keys in relational databases will also be discussed in this topic. You will also learn how to access databases from middleware or Node.js code which is the interaction of middleware and the data tier. You'll remember that this is part of the three-tier web application architecture I discussed earlier in this course. Passing variables to the data tier from middleware will also be discussed in a programming exercise. We're also going to talk about the interaction of middleware and frontend in Node.js and you will learn how to pass parameters from Node.js code to EJS template files. Crucially this topic is designed to prepare you for the midterm assessment. So good luck with your assessment.

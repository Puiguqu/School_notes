Some screen readers may require using CTRL in conjunction with the alt key In this topic, I will talk about more advanced querying in SQL. Min, max, sum, and count are aggregate functions in SQL. They let the programmer write statements running on a set of records in order to do some simple calculations. This might be calculating the sum or counting the number of records, or finding the minimum or maximum number in a set of records and returning one single number. These functions are quite useful for creating reports or performing simple calculations that are needed often. In the previous topic, you learned about join statements in SQL. In this topic, we are going to build on that in order to learn about two other forms of join statements in SQL, left and right join. Nested select, which is also called select-inselect, is the next advanced querying technique you will learn in this topic. You will learn how to incorporate a SELECT statement within another SELECT statement, and you will see some queries written this way. As always, there are lab instructions and programming assignments designed and prepared for you to practice all these types of advanced querying in SQL. Although there is a lot more to learn about databases and the web, this is our last topic on these two subjects.

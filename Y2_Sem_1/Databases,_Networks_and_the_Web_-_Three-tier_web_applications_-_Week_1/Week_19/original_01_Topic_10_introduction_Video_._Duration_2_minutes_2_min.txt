Some screen readers may require using CTRL in conjunction with the alt key Welcome to Topic 10. In this topic, I will talk about networking concepts. This course is called Databases, Networks, and the Web. So far you have learned the basics of relational databases and how to develop a dynamic web application with access to databases. You have learned about databases in the context of building web applications, as I promised you in the first topic. In this topic, you will learn more about computer networking, inter-networking technologies, and the Internet. The dynamic web applications we develop can be accessed by users through a web browser. Web browsers are software providing access to resources on the web or on the Internet. The question that I will answer throughout this topic is, how his communication between computers provided by the inter-connected net works of computers or the Internet? This week you will learn about packet switching, which was a great idea at the time and became the foundation of the Internet we know today. You will learn about the TCP/IP model and Internet protocol or stack. Layers in the TCP/IP model will be explained for you one by one, starting from application layer, then transport layer, then network layer, and finally link layer. You will learn about UDP and TCP, multiplexing and demultiplexing, and much more. There is lots to cover in this topic. Most people who study computer science will go on to develop software in the application layer of the Internet. Nonetheless, it is necessary for you to know the basics of networking. Understanding how computers are connected to each other through networking technologies, including the Internet, will help you to design better applications. This in-depth knowledge about the infrastructure is helpful when designing and developing software running in application layer.

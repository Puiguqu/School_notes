A web server is a program that uses HTTP (Hypertext Transfer Protocol) to serve files to users in response to their requests. It is a crucial component of a three-tier web architecture, which consists of a client, application server, and database server. A web server processes incoming network requests over HTTP on several other related protocols.

HTTP is a client-server application protocol that defines the rules for exchanging requests and responses between a client program (such as a browser) and a server program (such as a web server). The user agent sends a request from a given resource to the web server, which then locates or builds the resource and sends a response back to the client.

The World Wide Web project was initiated in 1990 by Tim Berners-Lee and Robert Cailliau. The project aimed to build a distributed hypermedia system with a similar user interface, allowing users to access data stored remotely on servers connected to a network. HTTP is an application-level protocol that enables users to make requests of resources located at remote servers.

Some key definitions related to web servers include:

* A web server is software or hardware dedicated to running set software that can satisfy World Wide Web client requests.
* A web server can contain one or more websites and processes incoming network requests over HTTP on several other related protocols.

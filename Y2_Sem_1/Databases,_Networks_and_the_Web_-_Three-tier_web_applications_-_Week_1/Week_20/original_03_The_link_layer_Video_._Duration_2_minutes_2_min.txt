You'll remember that the link layer is the lowest and the closest layer to the physical part of the network which we call the physical layer. Unlike the network layer, which we discussed in the previous lecture, the link layer only worries about getting data across one hop. The link layer incorporates a group of communication protocols, and each protocol is designed based on a specific medium in their physical layer underneath. Link layer protocols cover different mediums such as wire, Wi-Fi, and fiber optics and Ethernet. It's irrelevant to the link layer, whether they Internet exist or not, since it is only responsible for carrying the data across one hop based on the medium available. So the link layer is where the link between the logical and physical components of the communication network is made. As I mentioned earlier in this topic, the term Frames refers to a packet of data at the data link layer. Because frames are concerned with transmitting data in a point-to-point manner, the addressing mechanism used by frame is hardware-specific. The MAC address or Media Access Control address is used in the medium access control protocol, which is a sub-layer of the data link layer. MAC addresses are often assigned by manufacturers as a burned in address. They allow link layer devices to identify themselves on a network.

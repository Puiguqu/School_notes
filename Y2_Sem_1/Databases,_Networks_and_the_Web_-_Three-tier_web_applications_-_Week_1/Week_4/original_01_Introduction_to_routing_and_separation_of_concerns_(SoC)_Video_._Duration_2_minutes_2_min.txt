You will also learn how to consider separation of concerns, or SOC, in your web application by reorganizing it into separate files and folders. I will explain this important software design and development principle before you proceed to apply it in your code and to reorganize your web application. Since it is not good practice to include HTML code in your web server software or middleware, in gaining an understanding of SOC, you will also learn about separating HTML files and rendering them in your web application. In addition to simple HTML files, you will learn about template engines and their benefits of including them in your HTML code and presentation tier. Template engines will enable you to insert variables into the final output of the HTML file. This is where your middleware connects to the front end by passing variables in order to provide a proper presentation of your web application to end users. Template engines will also enable you to run some programming logic at runtime before sending the final HTML to the browser for display.

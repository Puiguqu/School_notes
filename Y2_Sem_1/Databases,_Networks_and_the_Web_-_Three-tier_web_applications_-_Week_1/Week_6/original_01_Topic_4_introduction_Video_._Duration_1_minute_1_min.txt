In this topic, you will learn how to do form handling and how to collect an access form data in the middleware. You will also be introduced to two of the most common HTTP request methods named get and post. All user interactions including input forms are part of the front and or presentation tier of an application. Input forms are one way to provide user interaction in a dynamic web application, which generate some data, which we need to handle or store. In this topic, first you will learn how to create a form using HTML code. And then you will consider how to display it to the users in the front end of your dynamic web application. Secondly, you will learn how to collect users inputs to that form, or in other words, the form data from the front end, and then how to access it in the middleware or the application tier. In future topics, you will learn how to pass your collected form data from the middleware to the back end or database tier. So in this topic, you will learn the interaction between front end, which includes forms and middleware.

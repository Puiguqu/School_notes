This course, CM1010 Introduction to Programming II, is designed for novice programmers. It serves as a continuation for students who have previously completed Introduction to Programming I. These prerequisites included basic programming techniques in JavaScript and the p5.js library. The core function of this course is to build upon existing knowledge. Learners will develop the ability to combine multiple short code segments into comprehensive, larger projects. A major technical focus is the development of object-oriented programming (OOP) knowledge. Specifically, students will learn foundational principles such as encapsulation and abstraction. The curriculum revolves around three main case study applications. These modules include a data visualiser, a drawing application, and a music visualiser. The course materials thoroughly outline the structure of each of these three applications. Students are guided to actively build their own custom features for these apps. Furthermore, learners will select one of these three case studies for independent extension. This final project requires students to apply advanced planning and organizational skills. Through this extended work, learners will master evaluating and refining their code structure. Upon successful completion, students will achieve the ability to write and evolve larger coding examples. They will also understand the underlying structure of a complex program. A crucial objective is programming using the fundamentals of object orientation. Learners will use advanced coding techniques to enhance efficiency and readability. Another key goal is the ability to test and debug larger, more intricate programs. The course emphasizes developing a broad knowledge of the profession of programming. No specific textbook is required for the successful completion of this course. However, several supplementary texts are recommended for further mastery. These books include *Getting Started with p5.js: Making Interactive Graphics in JavaScript and Processing* by Mccarthy, L., Reas, C., and Fry, B. (2016) from Maker Media, Inc. Additionally, *The Pragmatic Programmer: Your Journey to Mastery* by Thomas, D. and Hunt, A. (2020) is recommended, published by Addison - Wesley. Finally, *Coders at Work: Reflections on the Craft of Programming* by Peter Seibel (2009) from Apress provides supplementary reading material.

As with Introduction to Programming I, we are transitioning to a new code editor for this course: Visual Studio Code (VS Code). While our course has traditionally used Brackets, please note that Brackets is no longer supported by Adobe. Therefore, even if you can still use Brackets, we strongly recommend transitioning to VS Code for your coding activities.

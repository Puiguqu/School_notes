This second Introduction to Programming module aims to solidify prior knowledge and facilitate understanding of larger, real-world applications. A core focus area is the need for object orientation and dividing code into manageable, navigable chunks. Developing these skills will enhance the ability to interpret and organize complex codebases. The module culminates in a major project requiring the synthesis of various advanced programming elements. This project will be grounded in one of three foundational case studies presented over subsequent topics.

The program introduces three initial, bare-bones applications built using JavaScript and the p5.js framework. These case studies incorporate object-oriented techniques and other language features taught in Part 1. They are explicitly constructed to be extended with new features, tools, and functionality by the student. Students will begin by receiving guidance on the composition and structure of each application in separate topics. A key technical detail is that every case study utilizes constructor functions and demonstrates object orientation alongside a unique p5.js external library or object.

For initial understanding, students will complete an exercise for each case study, which will be peer-reviewed by classmates. Following this, the students are responsible for creating compelling creative extensions and new tools for the chosen application. Assessment criteria extend beyond mere technical implementation and code complexity. Crucially, students will be evaluated on their planning, organization, and evaluative skills used throughout the entire development lifecycle. This emphasizes that being an effective coder involves more than just coding ability.

The first case study, introduced in Topic 3, is a simple drawing application, similar to MS Paint or Apple's Mac Paint. This app uses the p5.js DOM library to access web page elements outside of the main Canvas for advanced controls and layout. Basic tools include drawing wiggly and straight lines, plus an airbrush tool, with potential for rich customization extensions.

In Topic 5, students will learn about the music visualization application, which utilizes the p5.js sound library. This library is powerful enough not only for music and sound playback but also for converting sound wave properties into measurable values. These values can then be used to generate sophisticated visual patterns, representations, and works of art.

Finally, Topic 7 introduces the data visualization application, an increasingly critical skill in journalism and public understanding. This module specifically uses the p5.js table object to effectively handle and manipulate tabulated data loaded from comma-separated value (CSV) files. After completing the case studies, the student must select one as the basis for their comprehensive project. Continuous guidance will be provided, including suggestions for extensions, progress tracking, and complex debugging techniques.

# Extending the drawing app: stamp tool

This tutorial demonstrates how to build a functional extension, specifically a stamp tool, for a case study drawing application. The goal is to create stand-alone sketches that demonstrate concepts, which users can then integrate into a provided template structure. Initially, the setup involves creating a canvas and an `index.html` file, along with an assets folder containing the sample image, `star.png`.

The first technical step is loading the image using the standard P5 function: `star = loadImage('star.png')`. The basic stamp functionality is implemented by checking the `if mouseIsPressed` variable within the `draw` loop. When the mouse is pressed, the function draws the image: `image(star, mouseX, mouseY)`.

Improvements were needed because the initial placement was incorrect and the image size was fixed. To correctly position the stamp, variables for sizing were introduced, such as `starWidth` and `starSize`. The drawing logic was refined using subtraction for positioning: `starX = mouseX - starSize / 2` and `starY = mouseY - starSize / 2`.

To elevate the functionality, the design was enhanced using user inputs via sliders (DOM principles). A variable `starSize` was established to dynamically control the stamp's diameter. Inside the `setup` function, a slider was created: `starSizeSlider = createSlider(5, 50, 20)`, with its parent set to a control element in the index.html. The star's actual size was then set using the slider's value: `var starSize = starSizeSlider.value`.

A second slider, `nstarSlider`, was added to control the number of stars, created as `nstarSlider = createSlider(1, 20, 5)`, similarly linked to the HTML structure. To draw multiple stamps, a `for` loop was implemented: `for (var i = 0; i < nstarSlider.value; i++)`.

Crucially, to prevent all stars from appearing at the exact same spot, randomness was introduced into the drawing loop. The coordinates were modified to incorporate randomized offsets: `startX = random(mouseX - 10, mouseX + 10)` and `startY = random(mouseY - 10, mouseY + 10)`.

This systematic addition of complexity demonstrates how the stamp tool, starting with a simple click function, can be built up with dynamic size control, quantity control, and positional randomness. This advanced extension serves as a model for creating rich, functional elements for the core application framework.

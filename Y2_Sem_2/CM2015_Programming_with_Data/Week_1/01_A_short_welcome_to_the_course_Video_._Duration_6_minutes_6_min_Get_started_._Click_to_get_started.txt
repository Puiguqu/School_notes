Data science is an important field spanning medicine, finance, and art, with McKinney's *Python for Data Analysis* defining it from descriptive to advanced machine learning statistics. The course defines "programming with data" as applying technical tools and libraries to prepare, clean, and transform data for statistical analysis. Unlike data science, which focuses on drawing conclusions, this discipline emphasizes structuring, organizing, and exploring raw data, a process known as data wrangling. Students should be familiar with Python or other programming languages to benefit fully from the curriculum. The core goal is mastering six key skills, starting with fundamental Python programming techniques. Core skills include writing programs with foundational Python concepts, utilizing development environments, and mastering purpose-driven libraries like NumPy, Pandas, and NLTK. These skills enable users to collect data from various sources, organize it logically, and form predictions. Furthermore, the curriculum mandates a holistic understanding of data, integrating reflection on social, legal, ethical, and professional issues surrounding data use. The first half of the course focuses on foundational Python programming skills, beginning with variables and control flow. Learners will progress to working with data structures, starting with lists and arrays, and using Python dictionaries to manage key-value associations. Technical coverage includes working with common file formats such as CSV and JSON, and specialized library usage for processing text-based data. By the second half, students conduct exploratory analysis of scraped web data, mastering data scraping, cleaning, transforming, and visualizing data to find correlations. The advanced topics culminate in using this processed data to make complex predictions. The evaluation system comprises a midterm coursework, requiring iterative chatbot development, and a final end-term examination, with each component contributing 50% of the total grade. Regular feedback is provided through un-graded formative assessments, including quizzes and programming labs. Throughout the curriculum, the syllabus follows a structured path, beginning with Jupyter Notebooks in Week 1 and advancing through data types and control flow in Week 3. Weeks 7 and 8 specifically address reading and writing data from file systems and databases, respectively. Content expands further through Week 9 and Week 10, focusing on processing and language processing of text data. Retrieval skills are deepened in Weeks 11 and 12 by covering web data retrieval, while Week 13 focuses on database data retrieval. Advanced techniques cover data cleaning in Week 15, followed by advanced data transformations and visualization in Weeks 16 and 17. Key external resources include *Python for Data Analysis* by Wes McKinney, *Automate the Boring Stuff with Python* for web collection, and specialized books for NLP: *Natural Language Processing in Action* (Howard and Lane), *Text Analytics with Python* (Sarkar), and *Natural Language Processing with Python* (Bert, Klein, and Loper).

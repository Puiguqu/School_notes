This module, CM2015 Programming with Data, is designed to equip students with comprehensive skills for working extensively with data. The course scope encompasses multiple critical data operations. These operations include acquiring data from various sources. Students will learn to visualize data in compelling and highly informative ways. Processing is key, allowing students to transform raw data into useful and shareable formats. Furthermore, the module emphasizes reasoning with data to test assumptions and explore complex datasets. A central technical element is the introduction of Python, a programming environment specifically adapted for these data applications. The learning methodology is activity-centric, focusing on hands-on exploration of data complexities. Through a variety of challenges, students will gain practical experience in technical data handling. Basic foundational elements covered include Jupyter notebooks, variables, and working with strings. Advanced programming concepts involve mastery of Lists, control flow, and defining functions. The curriculum explores foundational data structures necessary for organization. Practical skills include reading and writing data directly to the filesystem. Students will develop proficiency in processing raw text data. Data retrieval is taught in multiple contexts, including fetching data from the web. Additionally, the course covers methods for retrieving and utilizing data from various database sources. Data quality management is addressed through dedicated learning on cleaning and restructuring data. A significant component is data visualization, enabling effective graphical representation. The curriculum progresses through advanced topics in both Python and data programming practices. Upon successful completion, students will be able to set up and work within a Python environment for creating data-driven software. They will learn to write simple Python programs to perform robust data analysis and visualization tasks. Furthermore, graduates can explain how to retrieve data from various sources and correctly load it into a working program. Students gain the ability to prepare and effectively summarize complex data sets. Finally, they will acquire the critical skill to reason about data, making predictions and classifications.

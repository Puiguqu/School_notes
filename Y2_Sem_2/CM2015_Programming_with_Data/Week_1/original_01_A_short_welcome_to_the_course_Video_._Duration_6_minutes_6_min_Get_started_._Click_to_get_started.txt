In recent years, data science has emerged as an important area of study in computer science and influenced diverse areas such as medicine, health, finance, robotics and art. One of the recommended textbooks for this course, McKinney's Python for Data Analysis defines data science on a spectrum ranging from descriptive statistics to advanced statistical analysis in machine learning. But what does programming with data mean? Before data can be used for automated analysis or fed into machine learning models, it is prepared, cleaned, transformed into specific formats and explored through programming. In this course, I refer to programming with data as the application of programming techniques, libraries and tools to prepare data so that it can be used for statistical analysis. While data science focuses on analyzing data and drawing conclusions, data programming emphasizes on structuring, organizing and exploring data. I like to think of this process as solving a scrambled Rubik's cube. As data programmers, you will manipulate, reshape and organize the cube until it is in a well-ordered state. This process is commonly referred to as data wrangling. Students and professionals who are familiar with Python or another programming language and want to deepen their programming skills for building data-driven application and conducting analysis would benefit the most from this course. In this course, you will learn a range of topics in programming and data analysis, starting with writing interactive programs in Python to collecting raw data from web resources, progressing to analyzing and deriving insights from it. The six learning outcomes summarize the key skills you will learn by the end of the course. The first two learning outcomes concern the programming skills required in Python to handle data programming tasks. This includes writing programs with foundational programming concepts in Python, having a working knowledge of development environments in Python, and using a purpose-driven library such as NumPy, Pandas and NLTK for working with data. With these core skills, you will be able to collect from various sources like websites and databases, organize and reason about, as well as form hypotheses and make predictions using it. Finally, in addition to making you tech-savvy about data, we would also like you to have a holistic understanding of data and be responsible data programmers. This course is embedded with several discussion problems and other activities to make you reflect on the social, legal, ethical, and professional issues about collecting and using data from various sources. In alignment with the learning outcomes, the first half of the course will focus on programming skills in Python. Some of these concepts might already be familiar to you from earlier courses like Introduction to Programming. In the first few weeks, you will work with variables and write programs that involve control flow. You will also revisit data structures such as lists and arrays, which you may already know. In Topic 3, you will start using Python dictionaries to store associations between data entities. Storing associations in dictionaries is similar to storing key-value pairs in JavaScript objects, but dictionaries offer more features. In Week 4, you will learn about common formats for storing data like CSV and JSON and write programs to work with them. By Week 5, you will use specialized library for processing and analyzing text-based data. In the second half, you will build on these skills to conduct an exploratory analysis of data scraped from a web resource. Each topic will focus on a specific task related to data analysis topic, namely data scraping, cleaning and transforming data, plotting and visualizing it to find correlations in it. Towards the end of the course, you will use this data to make predictions. The evaluation for this module consists of two parts, the midterm coursework and the final end-term examination. Each of these contributes to 50% of your total grade. The midterm coursework involves weekly iterative development of a chatbot for submission at the midpoint of the course. The end-term examination includes all the topics covered in the course and will be conducted at the end of the course. To provide you with regular feedback on your progress, each topic includes formative assessments like quizzes and programming labs. While the formative assessments are not graded, they play an important role in your learning and performance on the course. I would encourage you to follow the weekly guidelines and make consistent incremental progress on your learning journey. To know the exact dates of the different assessments, please check the deadlines for these assessments on Coursera. The recommended textbook for this course is Python for Data Analysis, Data Wrangling with Python by Wes McKinney. For specific topics involving text analytics, I recommend the following books on natural language processing and Python. These are Natural Language Processing in Action by Howard and Lane, Text Analytics with Python by Sarkar, Natural Language Processing with Python by Bert, Klein, and Loper. For data collection from web sources, we will use Automate the Boring Stuff with Python. This is an online book and is also available in the reading resources. Once again, I welcome you to Programming with Data. I'm excited to guide you through this journey.

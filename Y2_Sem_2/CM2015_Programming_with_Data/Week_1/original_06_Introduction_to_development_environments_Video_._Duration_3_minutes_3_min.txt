Jupyter Notebook offers a facility to combine code and data to write interactive programs. Let's look at the organization of Jupyter Notebook and write our first Hello World program. What you see on the screen is an empty Jupyter Notebook. Pay attention to the title at the top, the menu, the toolbar, and the cells where you would write your programs. Cells in Jupyter Notebooks are organized into different types. By default, a cell is in the code type, which means you can write pieces of code in it and compile. Here, I'm going to write the first Hello World program using a print statement. Once you've written all that you wanted to write in a Jupyter Notebook cell, press Shift-Enter to compile and see the output of that. Other types of cells are available in Jupyter Notebook, such as markdown and raw cells. Once you select a cell, select the specific type to change the type of the cell to markdown or heading. A little more on markdown and raw cells. Markdown cells offer the facility to add documentation, which improves the literacy of the code. In markdown cells, you can write text as freeform. For instance, this is a freeform text in markdown. In markdown cells, you can also add titles and headings of different sizes. In order to do this, you would start the markdown with a hash symbol. There are six levels of hash symbols, which control the size and the boldness of the text that you add in the markdown. Let me quickly illustrate that. First, I would change the type of the cell to markdown. A single hash represents the largest size and the boldness of text. Adding six hash symbols makes the heading in the smallest size possible. In addition to controlling the size of the text, you can also add other things such as bullet points that help you indent the code. Markdown also offers a facility to add bullet points listed and unlisted. To create an unlisted bullet point, type the hyphen symbol followed by a space and then type the name of the bullet. To create a listed bullet point, add a number and then create the name of the bullet.

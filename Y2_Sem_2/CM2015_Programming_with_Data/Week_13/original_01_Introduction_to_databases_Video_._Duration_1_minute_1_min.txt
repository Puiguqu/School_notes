In a typical business setting, data might not be stored in text or CSV files, but in databases. SQL-based relational databases such as the SQL Server and MySQL are in wide use, as well as many alternative databases such as MongoDB have also become quite popular. The choice of database in an application is usually dependent on the performance, data integrity, and scalability needs. One reason relational databases in SQL have been widely used is the ease with which data can be joined, filtered, transformed, and aggregated. However, query languages like SQL impose certain limitations on the kind of group operations that can be performed on data. With the expressiveness of Python and Pandas, we can perform quite complex group operations that manipulate data interactively.

Simple merge Let's review the programming activity for merging data from CSV and SQL databases through worked examples. First, we will look at a simple merge where the data is organised with aligned indices. The SQL database and the CSV files are organised as follows. CSV file id name age 0 Alice 30 1 Bob 25 2 Charlie 35 3 David 40 4 Eva 28 SQL database id department salary 0 HR 50000 1 Engineering 70000 2 Marketing 60000 3 Finance 65000 4 Sales 55000 Note that the common field for both data is the 'id' key. While merging these, one way to combine the data is to merge all the columns that align with the id key. This line of code merges the containing data from the CSV file with the data from the SQL database on the id column. 1 merged_df = pd.merge(csv_df, sql_df, on='id') The result is a new data that includes all columns from both data, matched by the id values. The resultant output is: id name age department salary 0 Alice 30 HR 50000 1 Bob 25 Engineering 70000 2 Charlie 35 Marketing 60000 3 David 40 Finance 65000 4 Eva 28 Sales 55000 In some specific cases, you might need a more complex merge (e.g. handling different types of joins like left, right or outer joins). In this example, EmployeeID and ProjectID seem to be the primary keys to index the data. But, there is no match between the IDs which can facilitate a merge. The IDs that do merge between the two data are the EmployeeID and the LeadID.

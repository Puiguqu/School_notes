Data preparation is needed when real world data stored in files or databases is not in the right format for a particular task. Loading, cleaning, transforming, and rearranging are some common tasks that constitute data preparation. Such tasks are often reported to take up 80% or more of an analyst's time. Many researchers choose general purpose programming language like Python or C to process data from one form to another. In this course, we will use Pandas. Pandas along with built-in Python language features provides you with a high level, flexible, and a fast set of tools to enable data manipulation into the right form. In the videos in this topic, we will discuss techniques to prepare data by extracting it from unprocessed formats, organizing data in correct data types, handling missing and duplicate values in data, and other analytical transformations to filter outliers in data.

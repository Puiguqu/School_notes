# Enhancing your data with derived features
Reading•
. Duration: 45 minutes
45 min

URL: https://www.coursera.org/learn/uol-cm2015-programming-with-data/supplement/9dKgH/enhancing-your-data-with-derived-features

Identify some interesting features from scraped data using the data frame that you obtained from the programming lab and create new columns from this. Example transformation ideas: In a product database: Create a discount column that calculates the price after a discount based on existing price and discount rate columns. In customer feedback or product reviews: Create a sentiment or intent column based on the content of reviews, using keywords or sentiment analysis. For time-based data: Derive a time_of_day column by transforming timestamps into categorical values (morning, afternoon, evening). Now, try to implement it: Write a function for the transformation Use the map operation to apply it to a column and create new columns (refer to in-class code on map operations). You can apply similar transformations for multiple columns. For instance: Create a Price_Category column based on price ranges. Create a Review_Sentiment column based on the sentiment of product reviews. After you complete your transformations: Reflect on the process: How did creating new columns based on existing data enhance the dataset? What insights did you gain?

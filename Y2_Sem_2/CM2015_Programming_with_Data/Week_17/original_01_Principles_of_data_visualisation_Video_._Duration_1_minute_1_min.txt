Some screen readers may require using CTRL in conjunction with the alt key Making informative visualizations, sometimes called plots, is one of the most important tasks in data analysis. For some analysts, it may be a part of the exploratory process, for example, to identify outliers or the needed data transformation or as a way of generating ideas for models. For others, building an interactive visualization for the web may be the end goal. Python has many add-on libraries for making static or dynamic visualizations. In this topic, we will be mainly focused on matplotlib and libraries that build on top of it. Matplotlib is a Python package designed for creating plots and figures. Using matplotlib, you can assemble a plot from its base components, the data, the display type, i.e., the type of the plot, line, bar, box, scatter, contour, etc., legend, title, tick labels, and other annotations. Over time, matplotlib has spawned a number of add-on toolkits for data visualization. While matplotlib can be a really useful tool, it works best for smaller datasets such as data stored in lists. Libraries like pandas and seaborn provide built-in functions that are more suited to work with larger datasets. In this topic, you will learn the fundamentals of plotting using matplotlib and its integration with pandas.

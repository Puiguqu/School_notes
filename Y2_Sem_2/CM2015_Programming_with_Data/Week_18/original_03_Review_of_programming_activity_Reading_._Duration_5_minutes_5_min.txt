Text data often needs to be transformed into numerical values (such as counts or frequencies) for plotting or other analyses. A common mistake is trying to plot text directly without processing it into a more usable format. The following pie chart will not work because Pandas will attempt to plot the text as is, without counting the occurrences of each unique word. This can result in an empty or erroneous chart. 1 2 3 4 5 6 7 import pandas as pd import matplotlib.pyplot as plt # Data with raw text values df = pd.DataFrame({'Text': ['apple', 'banana', 'apple', 'apple', 'banana']}) # Plotting raw text df['Text'].plot(kind='pie') plt.show() Why this happens: Raw text is non-numeric, so plotting libraries like Matplotlib and Pandas require the data to be transformed into a numerical format (like counts) to visualise it properly. Solution: Convert the text data into counts (frequencies of each word) before plotting. 1 2 3 4 5 6 # Count the frequency of each word word_counts = df['Text'].value_counts() # Now, plot the pie chart with counts word_counts.plot(kind='pie') plt.show() Review of programming activity In this reading we will go through some common mistakes and challenges that can occur when conducting the text data conversion before plotting.

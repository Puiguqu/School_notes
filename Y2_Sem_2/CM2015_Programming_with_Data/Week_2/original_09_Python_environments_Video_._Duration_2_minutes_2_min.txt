To enter this environment, we use the conda activate command followed by the name of the environment. We know that the command is successful as cm2015 now appears before the interpreter. Now that we have created the command, let's install some relevant packages. To install a package, we use the pip install command. Pip is the default package manager in Python. Pip install followed by a name of a package will install the resulting package in this environment. One important thing to note is that the package installed is specific to cm2015 environment. It would not be available in other environments if you create any. So let's install numpy, matplotlib which are some common libraries that we will explore. Let's again try and install another package, in this case pandas. Now that pandas is installed, when we do a pip list, we see that the update contains pandas installed.

The first week, begin by understanding Jupyter Notebooks and running them on the Coursera platform. Jupyter Notebooks combine data and executable code in a single interface providing a useful facility to work on data programming problems. Notebooks will be the main programming environment for this course. On the programming side, you created and stored variables with integer, float, and string values and learned which types can be combined using arithmetic operations. Week two, further expanded on the first week to introduce the different components that form the Python language, namely packages and environments. Environments and packages were suggested that could be used to customize a local installation of Python and Jupyter on your computer. Finally, you worked on programs for string manipulation and concatenation, which are useful operations to extend strings to form longer patterns.

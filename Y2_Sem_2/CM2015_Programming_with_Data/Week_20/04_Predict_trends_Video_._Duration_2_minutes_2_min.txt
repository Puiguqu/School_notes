The final step in the learning process is Exploratory Data Analysis (EDA), which utilizes data understanding to make predictions. This activity is crucial for drawing inferences and answering complex questions about the dataset. The process is twofold: first, creating derived features, and second, applying linear regression to find correlations. It is highly advised to perform this activity only after completing all preceding parts of the notebook. This timing is critical because data predictions are extremely sensitive to existing data trends.

Creating derived features enhances the dataset by generating new variables from existing ones. For example, `discounted price` is derived continuously from the original `price` column. Furthermore, `affordable` is a Boolean feature, taking only true or false values, and is based on `price`. Another Boolean derivation is `is new`, which indicates if a product was released after a specified `launch date`.

A Boolean feature called `high` is generated by comparing the product’s rating against a specific threshold value. Finally, `sentiment` is a more advanced feature derived from the product `review` text. This derivation requires the use of regular expressions combined with sentiment analysis techniques. These derived features allow users to address questions, such as determining if new products tend to be affordable, or if affordable products generally possess high ratings.

Following this inquiry phase, the user must proceed to extract only two specific columns: `rating` and `price`. These two values are then utilized as the independent (X) and dependent (Y) variables for a linear regression model. The objective of applying linear regression is to calculate the correlation coefficient and assess the statistical significance of that correlation.

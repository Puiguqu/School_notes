They are equal to, greater than equal to, less than equal to, greater than, and less than. Let's apply these to different variables, namely integers or strings, and understand how they produce Boolean values. The first operator we will look at is equal to. Equal to is used to compare the equality of two values. When the two values are equal, the output of equal to is true. If the values are different, the value of equal to is false. Similarly, greater than or greater than equal to are used to compare whether one value is higher or lower than another value. So let's take the same values, a and b, and do a greater than comparison. If I type a greater than or equal to b, the output would be true. The output is true because the value of a is 6 and the value of b is 5. Lastly, the less than or equal to operator is used to compare whether the first variable is less than or equal to the second variable. Let's again take these two values, b equals 6, a less than or equal to b results in a true value, b less than or equal to a, conversely, results in a false value. Values of greater than, equal to, and less than are often used in expressions to evaluate whether they are true or false. Let's look at the operations that are available to combine multiple expressions to form a joint expression. These are three logical operators and, or, or not. The and operator is used in a condition to evaluate whether two things are simultaneously true. For instance, if I have two values, a equals 3 and b equals 6, I can use the and operator to simultaneously check if both these values are greater than 1. However, if one of these values is less than 1, then the and operation results in a false. So and operation simultaneously checks whether two conditions are correct. Different from the and is an or operator, which results in a true value even if one of the conditions is true. For instance, if I take the same example and change the and with an or, this would result in a true value as one of the values, which is b, is greater than 1. The last logical operation is a not operation, which gives the opposite of the true or false of a certain value. For instance, not of true is false and not of false is true. Going back to the same example, if I have a not before any greater than 1, then the result would be true because a is less than 1 and not of that is true. Two other operators that we are looking at are the identity operator and the membership operator. An identity operator is used to check whether two objects are the same. For instance, if I have a, which is a list of three values and b, which is a list of three values, identity operator used with the is symbol is used to check whether a and b are the same object. In this case, identity operator produces a false output because a and b are different objects as represented on the computer. However, if I create a third variable called c and assign it to a and use the identity operator, I would get a true value because a and c essentially point to the same object. The other operator we are looking at is the membership operator which is used with an in keyword. This is used to check whether a certain element is part of an object such as a list. For instance, let me take the value a as 3 and list which has three values 1, 2, 3. Using the in operator, I can check whether this value 3 is a part of this list. In this case, the value results in a true because a is indeed a part of the list. If I change the value to 5, the in operator results in a false value. The last operator that we are looking at is the is instance operator. This is used to check whether a certain variable belongs to a certain data type. For instance, I can use the is instance operator to check whether the value 5 belongs to the type int. In this case, 5 is an integer. However, if I change the value to hello, then is instance operator results in a false value as hello is an instance of type string and not integer.

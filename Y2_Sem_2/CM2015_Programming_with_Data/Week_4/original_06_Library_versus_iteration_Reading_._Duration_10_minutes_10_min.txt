Loops in Python are extremely useful when drawing patterns, such as triangles, trees or letters, using print statements. The key idea is to use loops to control how many lines are printed and where certain characters (like *, #, or @) are placed within those lines. By manipulating the loop's index, we can place characters at specific positions to form shapes. Here’s how loops can be used to draw some simple shapes: 1. Triangle: A triangle can be drawn by printing an increasing number of * symbols on each line. The loop helps control the number of stars printed per line. Example for a right-angled triangle: 1 2 for i in range(1, 6): # 5 rows print('*' * i) Output: * ** *** **** ***** 2. Tree: A tree can be drawn by printing a series of increasingly wider lines (stars) with a trunk below. 1 2 3 for i in range(1, 6): # 5 rows for leaves print(' ' * (5 - i) + '*' * (2 * i - 1)) # spaces for centre alignment print(' ' * 4 + '|') # tree trunk Output: * *** ***** ******* ********* | 3. Alphabetic letters can also be drawn with loops. Write a program that uses loops to print the letter "K" in a similar style. You can check your answer against the solution on the next page.

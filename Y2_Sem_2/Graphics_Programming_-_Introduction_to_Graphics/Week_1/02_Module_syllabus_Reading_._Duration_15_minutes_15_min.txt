# Module syllabus
Reading•
. Duration: 15 minutes
15 min - Summary

URL: https://www.coursera.org/learn/uol-graphics-programming/supplement/MlqiM/module-syllabus

This module, CM2030 Graphics Programming, is designed for programmers who have previously mastered the basics of the p5.js programming framework. It assumes prior knowledge gained through the "Introduction to Programming part one" course. The course begins with an intensive focus on simulating physical motion and dynamics on the screen. Initial learning involves building a self-contained physics engine from scratch. Specifically, students will learn to model movement using foundational concepts of vectors and forces. Following this foundational work, the curriculum transitions to utilizing pre-built, off-the-shelf physics libraries. After mastering basic physics simulation, attention shifts to the principles of procedural generation of graphics. This theoretical area is enriched by firsthand accounts from industry professionals. These speakers include both a working game designer and a computational artist. The module then undergoes a significant change in focus, moving into the realm of 3D graphics. The advanced study delves deeply into the fundamental building blocks of images: the pixels. Students will first grasp the basic techniques of pixel reading and direct manipulation. These skills are critical because they form the foundation for advanced image processing. Learners combine these core pixel techniques with other programming knowledge. The resulting capability is the ability to program sophisticated computer vision applications. Furthermore, the curriculum details the application of specialized image processing filters. The second part of the module culminates in a discussion with an accomplished artist-hacker. This expert illustrates how computer vision techniques are integrated into professional creative work. Practical learning is central to the module's design, featuring multiple weekly fun programming exercises. Knowledge retention is enforced through engaging programming assignments and detailed quizzes. Upon successful completion, students will gain the ability to manipulate numerical representations of visual media. They will also be able to explain the theoretical and mathematical principles underlying visual computing. A key objective is the ability to write basic but functionally complete graphics software systems. Moreover, learners will proficiently apply filters to transform images specifically for computer vision scenarios. The curriculum mandates that students explain the fundamental tenets of computer vision. Finally, students must write code that processes images to make intelligent assertions about their content. The course also teaches students how to select and effectively use various software libraries for software development. There is no official textbook, as the module is highly practical, video-based, and utilizes specific readings each week.

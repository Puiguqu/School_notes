This comprehensive guide details the core technologies and tools utilized throughout the course. A preliminary glossary has been provided to describe the core technologies that will be explained further in depth. Source code for every example demonstrated during class will be available in the "Weekly Source Code" section. Students are encouraged to download and thoroughly study this provided source code. The primary tool for development is p5.js, a JavaScript library running within the web browser context. p5.js is specifically designed to make coding accessible for artists, designers, educators, and beginners. It utilizes the metaphor of a software sketchbook, providing a full set of drawing functionality. Users are not restricted solely to the drawing canvas, as the entire browser page can function as the sketch. p5.js supports add-on libraries that facilitate interaction with various HTML5 objects. These objects include text elements, input fields, video streams, webcams, and sound. When coding a new example or replicating demonstration videos, the provided `emptyExample` must be used to ensure compatibility with the p5js library. The text editor showcased throughout the module was Brackets, which was used at the time of filming. However, Brackets is currently a discontinued editor and is only available via GitHub. Adobe has ceased maintaining the editor, though it remains open-sourced and accessible. The specific archived release of Brackets available is located at: `https://github.com/adobe/brackets/releases/tag/release-1.14.2`. Brackets itself is described as a lightweight, powerful, modern text editor. For more details on this program, one can refer to: `https://en.wikipedia.org/wiki/Brackets_(text_editor)`. If the user prefers an alternative, Visual Studio Code (VS Code) is strongly encouraged. VS Code is a free, open-source code editor. Users can download the application for Windows, Mac, or Linux using this link: `https://code.visualstudio.com/`. To replicate Brackets’ live preview functionality, VS Code offers a Brackets extension pack. After installing this package, a user can check their code in a browser using the "Go Live" button. The specific guide for this migration process is located at: `https://code.visualstudio.com/migrate-from-brackets`. Further documentation on this setup includes instructions titled: "Installing Brackets Extension for Visual Studio Code." Additionally, the course may require the occasional use of other technical libraries. These libraries include, but are not limited to, `matter.js` or various computer vision libraries. The instructions for embedding these external libraries will be demonstrated within the video content.

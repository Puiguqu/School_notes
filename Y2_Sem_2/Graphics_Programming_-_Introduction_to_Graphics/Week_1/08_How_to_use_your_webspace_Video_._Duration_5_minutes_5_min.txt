The primary objective is to share completed code safely online, ensuring that while others can view the results, the original solution code remains protected and prevents independent copying. The initial steps require downloading the designated "web folder zip," which must then be unzipped to reveal a directory containing all course assignments. These directories include formative and non-graded assignments, while graded assignments are strictly submitted to the team rather than shared publicly. For code that needs to be displayed, such as a test post sketch, the coder must first secure the source code using an obfuscator, specifically mentioning the "JavaScript obfuscator." This encryption process transforms the original sketch, replacing known code elements with obfuscated gibberish, thereby protecting the intellectual property. The resulting encrypted file (the index file) is the single, finalized representation of the entire sketch. Next, the user must place this encrypted index file into the appropriate subdirectory within the established web folder structure. Once the code is correctly placed, the entire web folder must be compressed into a zip file. This zip file is then uploaded using the "Open webspace uploader," a critical step in the deployment process. Users should ignore the initial warning about disallowed file types, as the system only accepts zip files for this purpose. After the upload, the user must navigate back to the instructions and click the "Generate and update circling" button. This button press initializes the system to process the newly uploaded code package. Following the generation, it is absolutely vital to visit the resulting generated link once; failure to do so will prevent the webpage from updating with the new code. The generated link must be copied and repeatedly called every time new code is uploaded to guarantee the webspace picks up and processes the latest zipped file. Once implemented, the link can be shared, allowing others to view the showcased work without accessing the underlying, protected source code. This meticulous process ensures the integrity of the solutions while facilitating collaborative viewing. The overall course structure covers a range of advanced topics, including 2D transformations, OOP in Javascript, vectors and animation, and intensive computer vision topics spanning multiple weeks.

# Checklist of required skills and knowledge
Reading•
. Duration: 10 minutes
10 min

URL: https://www.coursera.org/learn/uol-graphics-programming/supplement/ShOgC/checklist-of-required-skills-and-knowledge

To successfully complete the Coursework 1 Assignment, you should master specific skills, concepts, and knowledge as outlined below: Introduction to Graphics, Vectors, and Animation Key Concepts: 2D transformations and their application in animations. Basic vector concepts and how they influence movement and direction. Introduction to object-oriented programming for creating modular, reusable code. Skills and Knowledge: 1. Understanding 2D Transformations: Mastery in scaling, rotating, and translating objects in 2D space Applying transformations to create movement and position objects dynamically. 2. Vectors and Vector Arithmetic: Knowledge of vector properties and basic operations (addition, subtraction, dot product). Practical application in object motion, such as programming a clock or solar system. 3. Basic Programming for Animation: Using programming concepts to animate transformations, such as rotating clock hands or orbiting planets. Understanding the loop mechanism in programming to animate objects continuously. 4. Object-Oriented Concepts (Classes): Comprehension of how classes and objects can encapsulate data and functionality. Using classes to model and simulate systems (e.g., a solar system). Learning Outcomes: Explain and implement 2D transformations. Describe and use classes in programming. Program basic systems (solar system, clock) using transformations and vectors. Forces and Physics Key Concepts: Force applications in programming animations. Basics of using a physics engine, such as matter.js, for simulating forces and interactions. Skills and Knowledge: 1. Physics Concepts in Motion: Understanding forces such as gravity, friction, and collision mechanics. Applying these forces within animation scenarios to create realistic motion. 2. Particles and Physics Libraries (Matter.js): Using physics libraries to simulate real-world forces and interactions between objects. Implementing simple particle systems to visualize concepts like collision and attraction. 3. Collision Detection and Response: Coding techniques to detect collisions between objects. Implementing basic responses (e.g., bouncing or stopping objects upon impact). Learning Outcomes: Explain forces and their role in animation. Use and configure matter.js for animation. Implement basic physics systems in programming scenarios. Physics with matter.js and Generative Techniques Key Concepts: Deeper exploration of physics engines and how they simulate complex interactions. Introduction to generative art through procedural techniques like noise and randomness. Skills and Knowledge: 1. Advanced Matter.js Usage: Understanding bodies, constraints, and composites within matter.js. Creating interconnected objects that simulate physical properties and constraints. 2. Generative Art and Design: Applying randomness and noise to create dynamic, generative systems. Implementing art techniques that respond to algorithmic or randomized inputs. Learning Outcomes: Define generative art and its characteristics. Apply randomness in generative systems. Implement complex systems using matter.js and generative techniques. Generative Techniques: Randomness, Noise, and Fractals Key Concepts: Recursive patterns and fractals for creating detailed, repetitive designs. Trigonometric functions to achieve oscillatory movement and intricate shapes. Skills and Knowledge: 1. Fractals and Recursive Systems: Understanding recursion and its role in creating fractal patterns. Creating fractal systems by implementing recursive functions. 2. Trigonometry and Oscillation: Using trigonometric functions to create shapes and movement. Coding oscillating patterns to produce wave-like motions in animations. 3. Additive Synthesis in Animation: Implementing additive synthesis techniques to overlay motions and patterns. Combining multiple simple animations to form complex generative systems. Learning Outcomes: Use trigonometry to create and animate shapes. Implement recursive fractal systems. Develop generative art systems using noise and randomness. Generative Techniques and Procedural Generation in Games Key Concepts: Procedural generation and its applications. Techniques to create complex visuals in gaming through generative art. Skills and Knowledge: 1. Procedural Generation Techniques: Understanding procedural generation and its application in games. Evaluating the benefits and limitations of using procedural methods. 2. Applying Art Techniques: Applying polar coordinates and oscillatory functions to create designs. Using procedural methods to develop generative patterns within game contexts. 3. Linking Course Content with Professional Insights: Analysing Andy Lomas’ work to identify practical applications of generative techniques. Connecting learning outcomes to real-world industry practices. Learning Outcomes: Explain procedural generation’s use in gaming. Implement generative systems in animations using polar coordinates. Identify how generative design can be used professionally.

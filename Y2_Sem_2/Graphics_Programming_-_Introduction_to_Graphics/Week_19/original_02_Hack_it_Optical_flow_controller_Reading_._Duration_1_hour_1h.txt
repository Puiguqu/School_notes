opticalFlowController-Template Use the opticalFlowController-Template to control graphics on screen with the movement of your limbs or the movement of the camera, as in the image above. At the bottom of the sketch.js file is the genRings() function which creates these generative rings. Notice how we pass to the genRings() function the movement on the x-axis (somewhat constrained) in order to make the rings move in and out. Things to try Modify the code inside genRings() to also accept the movement on the y-axis. Have the movement on the y-axis affect the colour of the rings. Create your own generative graphics function using code from earlier classes in this course or by adopting code you found on https://www.openprocessing.org/ Make sure your graphics are controlled both by the movement on the x-axis as well as the movement on the y-axis. Record your creation as a short gif and post it on the forum!

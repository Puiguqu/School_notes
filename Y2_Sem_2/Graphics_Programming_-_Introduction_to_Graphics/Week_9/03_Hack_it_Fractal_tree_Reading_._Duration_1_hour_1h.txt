# Hack it: Fractal tree

. Duration: 1 hour
1h - Summary

URL: https://www.coursera.org/learn/uol-graphics-programming/supplement/QVpaB/hack-it-fractal-tree

This project guides the user through enhancing a basic fractal tree using recursive coding techniques. Initially, the user must download and thoroughly study the provided basic fractal tree template code. A primary modification involves changing the sub-branch length ratio from a static 67% of the parent branch. To introduce variation, this ratio must be randomized within a specific range of 0.5 to 0.8. When implementing random functions, repeated calls within the `draw()` loop should be avoided. The setup function uses `randomSeed()` to ensure the sketch maintains consistent values across refreshes. Users are encouraged to test the function's behavior by commenting out the `randomSeed()` call. The angular setting for the next branch must be randomized, rather than fixed at the static 45 degrees, to achieve asymmetry. The trunk should be colored dark brown, gradually lightening the color of the branches as the drawing progresses upward. A crucial mechanism involves checking the branch size near the end of the `branch()` function. If the branch falls below a certain minimum threshold, the recursive subdivision process must halt. Instead of continuing the recursion, an "else" statement must be added to draw a leaf in the appropriate location. These replacement leaves should be drawn in ellipses, utilizing plausible and interesting color schemes instead of purely random colors. For advanced enhancement, wind effects can be added by incorporating the `noise()` function. This function should add a specific amount to the calculated random angle within the `branch()` function. Finally, understanding particle concepts allows for simulating the natural event of leaves falling at random intervals. The entire process heavily relies on understanding and manipulating recursive operations within the programming environment. These modifications deepen the realism and complexity of the generative fractal artwork. The structure of the code relies on a dedicated `branch()` function to manage the recursive calls. Students are expected to play with the code to understand the foundational structure of the tree generation. The goal is to transform a basic diagram into a detailed, naturalistic fractal representation.

Understanding of Generative Art and Design I can explain the basic principles of generative art and design. I understand how generative systems can create complex, dynamic visuals. 2. Use of Polar Coordinates I am comfortable using polar coordinates to position elements in my designs. I can translate between polar and Cartesian coordinates in my code. 3. Application of Trigonometry for Shape Creation I can use sine and cosine functions to create circular and oscillating patterns. I have applied trigonometry to generate complex shapes and forms. 4. Implementation of Oscillation for Movement I can explain oscillation and how it can produce movement in a visual design. I can implement oscillation in code to create continuous, smooth motion in generative art. 5. Using Additive Synthesis in Generative Systems I understand additive synthesis and its role in creating generative systems. I can apply additive synthesis techniques to combine multiple elements in my designs. 6. Creation of Recursive Systems for Fractals I understand recursion and its importance in creating self-repeating patterns. I can implement recursive functions in code to produce fractal designs. 7. Overall Application of Generative Techniques I can combine sine waves, oscillation, and recursion to develop complex generative art. I am able to experiment with different parameters to achieve unique generative designs.
